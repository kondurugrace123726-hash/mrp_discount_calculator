# MRP Discount Calculator

This is a simple Python program that calculates the final price of a product after applying a discount on the MRP (Marked Price).

## 🚀 Features
- Takes MRP and discount percentage as input
- Calculates discount amount
- Displays final payable price
- Output formatted to 2 decimal places

## 📥 Input
- MRP (integer or float)
- Discount percentage

## 📤 Output
- Final price after discount

## 🧮 Formula
Final Price = MRP - (MRP × Discount / 100)
